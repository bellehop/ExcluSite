Frequently Asked Questions (FAQ)

How Does the Blacklisting Feature Work?

Search results and sponsored content are hidden based on the domains in your blacklist. You can add, remove, and edit domains in the blacklist freely.

How Do I Edit the Blacklist?

Click the 'Edit' button, edit the domains in the Blacklist as needed, and click the 'Save' button to apply your changes.

What does the Logger Terminal do?

The Terminal in the popup shows real-time updates, including number of filtered search results, the names of filtered results, and the number of filtered sponsored ads.

How Should I Input Websites/Domains?

When adding websites to the blacklist, please follow these guidelines:

  • Domain Format: Input the domain of the website without the "http://", "https://", or "www." prefixes. For example, use "example.com" instead of "https://www.example.com".
  • No Trailing Slash: Do not include a trailing slash at the end of the domain. For example, use "example.com" instead of "example.com/".
  • Geovariants: If a website has different geovariants (like google.com, google.co.uk, google.co.in, etc.), each variant should be added separately to the blacklist. The extension treats each geovariant as a separate domain.
